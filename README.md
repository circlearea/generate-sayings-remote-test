## Installation
1. clone project to your local computer
2. open a command line interface
   1. install node.js from nvm
   2. input : npm init -y
   3. input : npm install
   4. input : npm i -g nodemon
   5. input : npm run dev
3. open a browser and input url: http://localhost:3000
